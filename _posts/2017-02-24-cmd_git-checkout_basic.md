> Go to a previous commit
> 
    $ git checkout <hash>

<div></div>

> Go to a tag
> 
    $ git checkout <tag>

<div></div>

> Go back to last commit
> 
    $ git checkout          # or
    $ git checkout master

<div></div>

> Go back to `HEAD` for a file
> 
    $ git checkout <file>
