> Go to parent commit of a previous commit
> 
    $ git checkout <hash>^

<div></div>

> Go to parent commit of a tag
> 
    $ git checkout <tag>^

<div></div>

> Go to "grand parent" of a tag/commit
> 
    $ git checkout <object>^^

<div></div>

> Create a branch and navigate to it
> 
    $ git checkout -b <new_branch>
