> Add a tag to `HEAD`
> 
    $ git tag <tagname>

<div></div>

> Add a tag to a specific commit
> 
    $ git tag <tagname> <hash>

<div></div>

> Delete a tag
>
    $ git tag -d <tagname>

<div></div>

> List all branch tags
> 
    $ git tag
